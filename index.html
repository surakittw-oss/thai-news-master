3 โหมดการทำงาน

บก. แปลข่าว — แปลข่าวต่างประเทศเป็นไทย ใช้คำตามราชบัณฑิตยสถาน เรียบเรียงด้วยโครงสร้างหัวข้อย่อยแบบบทความเชิงลึก

เขียนข่าว — เรียบเรียงข่าวจากข้อมูลที่ให้ ครอบคลุมทุกหมวด ทุกภาษา

ทำ SEO — รับบทความสำเร็จ → สร้าง SEO/GEO/AEO metadata ครบชุดให้

โมเดลที่รองรับ

OpenAI GPT-4o · Google Gemini 2.5 Flash · Anthropic Claude Sonnet 4.5 — ใส่ key แต่ละเจ้าใน "ตั้งค่า"

โหมดเปรียบเทียบ

ติ๊ก "เปรียบเทียบหลายโมเดลพร้อมกัน" → เลือกหลายโมเดลได้ → ระบบเรียก API ทุกตัวพร้อมกัน เทียบผลลัพธ์ side-by-side

โครงสร้างบทความ

บทความที่ AI เขียนจะมี: หัวข่าวที่ดึงดูด, ย่อหน้านำ (lead), หัวข้อย่อย 3-5 ข้อพร้อมรายละเอียดเชิงลึก, สรุปท้ายเรื่อง, URL Slug, Keywords, อ้างอิง, คำบรรยายปกข่าว และ SEO/GEO/AEO ครบชุด

URL fetching

เบราว์เซอร์บล็อก CORS ของสำนักข่าว — ก๊อบเนื้อหามาวางตรงๆ ดีที่สุด

ความเป็นส่วนตัว

API key เก็บใน localStorage ของเบราว์เซอร์ เนื้อหาที่วางจะถูกส่งไปยัง provider (OpenAI/Google/Anthropic) ตาม policy ของแต่ละเจ้า
