3 โหมดการทำงาน

บก. แปล EN→TH — แปลข่าวต่างประเทศเป็นไทย ใช้คำตามราชบัณฑิตยสถาน เรียบเรียงด้วยโครงสร้าง Lead → Context → Why it matters → What's next

เขียนข่าว — เรียบเรียงข่าวจากข้อมูลที่ให้ ครอบคลุมทุกหมวด ทุกภาษา

ทำ SEO — รับบทความสำเร็จมา สร้าง SEO/GEO/AEO metadata ครบชุดให้

โมเดลที่รองรับ

OpenAI GPT-4o, Google Gemini 2.0 Flash, Anthropic Claude Sonnet 4.5 — ใส่ key ของแต่ละเจ้าใน "ตั้งค่า"

โหมดเปรียบเทียบ

เลือกหลายโมเดล + ติ๊ก "เปรียบเทียบหลายโมเดลพร้อมกัน" → ระบบเรียก API ทุกตัวพร้อมกัน เทียบผลลัพธ์ได้ side-by-side

URL fetching

เบราว์เซอร์บล็อก CORS ดึง URL ของสำนักข่าวต่างๆ ไม่ได้ — ให้ ก๊อบเนื้อหาข่าวมาวางตรงๆ แทน

ความเป็นส่วนตัว

API key เก็บใน localStorage ของเบราว์เซอร์เท่านั้น เนื้อหาที่คุณวางจะถูกส่งไปยัง API ของ provider ที่เลือก (OpenAI/Google/Anthropic) ตาม policy ของแต่ละเจ้า
